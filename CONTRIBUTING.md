# Contributing

Contributions are welcome! If you'd like to suggest a feature, report a bug, or submit a fix, feel free to open an issue or pull request.

## Getting Started

1. Fork the repository and create a new branch for your changes
2. Make your changes and test them locally
3. Open a pull request with a brief description of what you changed and why. If you added new dependencies, please mention them as well.

## Guidelines

- For significant changes, please open an issue first to discuss the idea
- Keep pull requests focused, one fix or feature per PR is easier to review
- Update the README.md if your changes affect usage or features

## Code of Conduct

Be respectful and constructive. This is a hobby project and a learning space — feedback and contributions should be kind and collaborative.
